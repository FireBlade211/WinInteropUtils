LengthDrawn

Receives the number of characters processed by DrawText(Graphics, string, ref Rectangle, ThemeTextOptions, ref DrawTextAdvancedOptions), including white-space characters. The number can be the length of the string or the index of the first line that falls below the drawing area. Note that DrawText(Graphics, string, ref Rectangle, ThemeTextOptions, ref DrawTextAdvancedOptions) always processes the entire string if the NoClip formatting flag is specified.
