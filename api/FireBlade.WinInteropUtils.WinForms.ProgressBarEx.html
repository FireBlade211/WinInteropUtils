PerformStep()

Advances the current position of the ProgressBarEx by the amount of the Step property, and redraws the control to reflect the new position.

public int PerformStep()

Returns

int

The previous position.

Remarks

When the position exceeds the maximum range value, this method resets the current position so that the progress indicator starts over again from the beginning.

Exceptions

InvalidOperationException

A step cannot be performed on a marquee progress bar as it does not use a value.

InvalidOperationException

Can't perform a step right now; the control handle has not been created yet.
