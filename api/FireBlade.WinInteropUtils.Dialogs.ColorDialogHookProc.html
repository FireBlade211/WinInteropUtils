Defines the hook procedure for a ColorPickerDialog.

public delegate nuint ColorDialogHookProc(nint hDlg, uint uMsg, nuint wParam, nint lParam)

Parameters

hDlg nint

A handle to the Color dialog box for which the message is intended.

uMsg uint

The identifier of the message being received.

wParam nuint

Additional information about the message. The exact meaning depends on the value of uMsg.

lParam nint

Additional information about the message. The exact meaning depends on the value of uMsg.

Returns

nuint

If the hook procedure returns zero, the default dialog box procedure processes the message.

 If the hook procedure returns a nonzero value, the default dialog box procedure ignores the message.

Remarks

If the uMsg parameter indicates the WM_INITDIALOG message, then lParam is a garbage collector handle pointing to the instance of the ColorPickerDialog class. To get the instance, call the FromIntPtr(nint) function and pass in the given lParam.
